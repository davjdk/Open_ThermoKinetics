```python
            safe_msg = msg.replace("%", "%%")  # –≠–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞–Ω–∏–µ %
            try:

        return str(msg)
    except Exception:
        return f"[Message formatting error: {type(e).__name__}: {e}]"
```
